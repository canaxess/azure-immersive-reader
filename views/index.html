Azure Immersive Reader

This demo shows how the Immersive Reader tool could be used to assist a users literacy and comprehension. Article: Using Azure to help users with lower levels of literacy

Websites may have sections of text at an advanced reading age, the tool could help a user understand individual words, sections of text or the entire content.

Additionally, it could be used as technique to pass more significant WCAG 2.1 success criteria.

  1. Highlight all or some of the text
  2. Select the 'Launch Reader' button
